# Media & Democracy Simulation

## Description of the Project
The **Media & Democracy Simulation** is an interactive, serverless educational web portal and simulation game. It is designed to teach citizens and students how the press operates in a democracy, the distinct roles it plays, and how to practice critical media literacy in their daily lives.

## Problem Statement
In the modern digital age, citizens are constantly bombarded with sensationalized headlines, misinformation, and unverified rumors. This clutter damages public trust, weakens democratic transparency, and creates hyper-polarization. Many users struggle to identify biases, check sources, or understand how their sharing habits impact the health of society's democratic institutions.

## Solution (What your project does)
This project provides a hands-on, engaging platform that combines structural learning with an interactive simulation:
1. **Structural Education**: Explores the 8 key roles that the media plays in a free democracy and details a 3-step checklist to spot media bias and misinformation.
2. **Interactive Simulation**: Puts users in the hot seat with the "Headline Challenge Arena," where they evaluate 5 random scenarios out of 25 potential stories. 
3. **Consequence Visuals**: Instantly reveals the impact of their choices (Verify, Share, Ignore, or Report) on personal and community civic health, demonstrating how critical checking is to protecting the truth.

## Features
* **Interactive Media Roles Guide**: Deep dive into the 8 core democratic roles of the press:
  * *Watchdog*: Exposing corruption and holding power accountable.
  * *Informer*: Providing unbiased, verified facts for public decisions.
  * *Public Forum*: Amplifying diverse community voices and opinions.
  * *Gatekeeper*: Verifying and filtering rumors before publication.
  * *Agenda Setter*: Focusing public attention on critical, ignored issues.
  * *Mobilizer*: Encouraging voting and civic engagement.
  * *Civic Educator*: Explaining dry, complex legislation clearly.
  * *Consensus Builder*: Creating compromise and bridging polarization.
* **Media Literacy Checklist**: Covers three core pillars: identifying bias, checking sources, and cross-referencing information.
* **Headline Challenge Arena**: Play 5 random rounds generated from a pool of 25 distinct scenarios.
* **Democracy Impact Dashboard**: Dynamic, real-time outcomes displaying personal civic health change, simulated global community votes, and a community resilience index.
* **Citizen Report Card**: Generates a round-by-round breakdown of the game showing your selected options, correct responses, and civic health rankings.

## Tech Stack Used
* **Languages**: HTML5, CSS3, JavaScript (Vanilla ES6)
* **Assets**: Lucide Icons, Google Fonts (*Outfit* & *Inter*)
* **Deployment**: Netlify (Static & Serverless)

## How to Run the Project
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/Vikas-4777/Swechaap-S02-.git
   cd Swechaap-S02-
   ```
2. **Local execution (No Server Required)**:
   * Double-click `index.html` to open the application directly in any web browser, or host it locally using a simple VS Code Live Server extension.
3. **Local Node server (Optional)**:
   ```bash
   npm install
   npm start
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser.
